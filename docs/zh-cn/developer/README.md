# Developer

F2 提供了多种方式便于开发者扩展 F2 的功能。

# Previous versions

F2 3.1 版本同之前的 3.0 版本主要有以下区别：

1. `chart.guide()` api 参数类型发生变化；
2. 底层绘图引擎改造，原先 `F2.G` 中的所有方法全部废弃，新版 G 的使用参见[ graphic](./graphic.md)；
3. 动画接口变更，详见 [Animation API](../api/animation.md)
